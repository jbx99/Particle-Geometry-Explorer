# Particle Geometry Explorer

![Particle Geometry Explorer Overview](img/pg-overviews.jpg)

A sophisticated, interactive 3D particle simulation built with Three.js. This application allows users to explore complex geometric shapes, orbital systems, and dynamic motion effects in a visually stunning environment.

### ‚¨° Particle System
*   **Diverse Shapes**: Choose from basic geometries (Sphere, Cube, Tetrahedron, etc.) to complex forms like Torus, DNA Helix, and even custom text.
*   **Custom Sprites**: Render particles as Circles, Stars, Hearts, Happy Faces, and more.
*   **Scalability**: Render up to 10,000 particles with adjustable system size and density.

### ‚ú® Motion & Effects
*   **Dynamic Effects**: Apply real-time effects like Pulse, Wave, Spiral, Breathe, and Explode.
*   **Time Control**: Adjust simulation speed, rotation, and animation delays.
*   **Interactive Physics**: Particles react to mouse interaction (Attract/Repel modes).

### üåÄ Orbital Rings
*   **Complex Systems**: Create multi-ring orbital systems with up to 5 independent rings.
*   **Presets**: Quickly load configurations like Atom, Solar System, or Galaxy.
*   **Customization**: Fine-tune ring size, speed, orientation (XY, XZ, YZ, Gyroscope), and tilt.

### üé® Visual Customization
*   **Color Palettes**: Switch between themes like Fire, Ice, Forest, Neon, and Cyberpunk.
*   **Backgrounds**: Choose from Solid, Starfield, Nebula, Cyberpunk Grid, and more.
*   **Visual Enhancements**: Toggle Particle Glow, Trails, and Connection Lines.

*   **Demo Mode**: Auto-randomize settings for a hands-free visual experience.

*   **Diagnostics**: Monitor performance with a detailed FPS and system stats panel.

## üéÆ Controls

### Mouse & Touch
*   **Left Click + Drag**: Rotate camera around the scene.
*   **Right Click + Drag**: Pan the camera.
*   **Scroll Wheel**: Zoom in/out.
*   **Touch (1 finger)**: Rotate.
*   **Pinch (2 fingers)**: Zoom.
*   **3 Finger Drag**: Pan.

### Keyboard Shortcuts
*   **Space**: Pause/Play animation.
*   **S**: Take a screenshot.
*   **R**: Randomize settings.
*   **C**: Reset camera position.
*   **M**: Lock/Unlock camera movement.
*   **D**: Toggle diagnostics panel.
*   **F**: Toggle fullscreen mode.
